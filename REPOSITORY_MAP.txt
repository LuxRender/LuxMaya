What's in this respository ?
----------------------------


./installer/																	Installer scripts

	linuxInstallerSrc.sh														Linux installer bootstrap script
	linuxInstallScript.sh														Linux installer payload script
	makeDist.sh																	Distribution package maker
	windowsInstaller.nsi														Windows installer NSIS script

./src/																			MEL Scripts, Icons and other package files.

	*.txt																		Info included in distribution packages
	Mel/*																		LuxMaya MEL scripts
	Icons/*.xpm																	LuxMaya Icons
	Icons/png/*																	LuxMaya Icons in PNG format
	Icons/src/*																	LuxMaya icon srouce files. SVG + PSD.
	attrPresets/script/*														LuxMaya Render settings presets

./matlib/																		Material library/Lux online integration

	safeeval.py																	Safe Eval() class
	webfetch.py																	Class to fetch preset data fom the web
